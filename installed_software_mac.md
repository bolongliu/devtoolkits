# 使用Gemini CLI 分类整理系统软件

```bash
1. geminni
2. 帮我 获取 mac上安装的所有软件 列表，最好能 做 分类。输出md格式，并生成对应文件。注意忽略系统软件。
3. 将生成的文件里面的软件分为合理的类别，并且每个软件后面用一句描述其核心功能。
```
